# RS Automotive - Site Institucional

Este é o site institucional da RS Automotive, uma estética automotiva premium em Guarulhos.

## Tecnologias

- **HTML5**: Estrutura semântica.
- **CSS3**: Estilização com layout responsivo e design premium.
- **JavaScript**: Lógica de animação e interação.
- **GSAP**: Biblioteca de animação de alto desempenho (carregada via CDN).

## Como Visualizar

Basta abrir o arquivo `index.html` em qualquer navegador moderno. Não há necessidade de ferramentas de compilação ou instalação de dependências.
